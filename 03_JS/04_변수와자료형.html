변수와 자료형

변수 선언 var, let, const

변수 선언법

  • var : 중복 선언 가능, 재할당 가능
  • let : 중복 선언 불가능, 재할당 가능
  • const : 중복 선언 불가능, 재할당 불가능

* 자료형

변수 선언 시 자료형을 별도로 지정하지 않음 => 자료형 개념이 없지는 않음!!
한 줄씩 실행되는 인터프리터 언어 특성상 변수에 대입되는 값에 따라서 알아서 자료형이 결정

자바스크립트의 자료형

  • 숫자 : Number
  • 문자열 : String
  • 논리값 : Boolean
  • 초기화하지 않은 값 : undefined
  • 값이 없는 것 : null
  • 객체/배열/함수 : Object

* 변수 명명 규칙 *

  1. 일반적으로 영어(소문자/대문자)를 사용하고, 문자와 숫자를 모두 사용 가능
  2. 특수문자 _또는 $만 사용 가능
  3. 숫자로 시작할 수 없음 ex) 100 (x), 1abc (x)
  4. 키워드(예약어)를 변수명으로 사용할 수 없음 ex) if, let, const, for ...

표기법 (Naming Type)

  1. camelCase : 자주 사용되는 표기법 소문자로 시작하고 다른 단어가 시작될때마다 대문자로 구분
    • user id -> userId
    • product name -> productName
  2. snake_case : 파이썬 표기법. 단어마다 _로 구분
    • user_id
    • product_name
  3. PascalCase : c# 또는 ms계열의 언어 표기법. 모든 단어의 시작을 대문자.
    • UserId
    • ProductName
