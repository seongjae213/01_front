속성 선택자

  • 요소에 있는 속성을 이용해서 선택
  • 선택자 뒤에[]를 이용하여 속성,속성값을 제시
        *키워드 : 속성값

        키워드가 일치하는 경우 : 선택자[속성명=키워드]
        키워드가 포함되어 있는 경우 (전체단어) : 선택자[속성명~=키워드]
        키워드가 포함되어 있는 경우 : 선택자[속성명*=키워드]
        키워드로 시작하는 경우 (전체단어 또는 단어 다음에 -가 있는 경우) : 선택자[속성명|=키워드]
        키워드로 시작하는 경우 : 선택자[속성명^=키워드]
        키워드로 종료하는 경우 : 선택자[속성명$=키워드]

자손 선택자와 후손 선택자

  • 중첩된 요소들 기준으로 선택 가능
  • 자손 : 바로 하위에 있는 요소들
  • 후손 : 하위 요소 모두(전부)
        *자손 선택자 : >
        a > b {
            스타일속성 : 값;
        } /* a요소의 자손 요소 b */

        *후손 선택자 : 공백
        a b {
            스타일속성 : 값;
        } /* a요소의 후손 요소 b */

동위(같은 레벨) 선택자

  • 동위 관계(같은레벨)에 있는 특정 요소를 선택할 때 사용
  • 선택되는 요소가 뒤에 위치하는
        * a 요소 바로 뒤에 있는 b 요소 하나만을 선택
        a + b{
            스타일속성: 값;
        }

        * a 요소 뒤에 모든 b 요소 선택
        a ~ b {
            스타일속성: 값;
        }

상태 선택자

요소의 상태에 따라 선택되는 선택자

(1) 체크된 상태의 요소

        선택자:checked{
            스타일속성: 값;
        }

(2) 초점이 맞춰진 요소

        선택자:focus {
            스타일속성: 값;
        }

활성 / 비활성 요소 선택자

        *활성화된 요소
        선택자:enabled{ 스타일속성: 값; }

        *비활성화된 요소
        선택자:disabled{ 스타일속성: 값; }

반응 선택자

사용자의 움직임에 따라 선택되는 선택자

        *해당 요소가 클릭되었을 경우
        선택자:active{ 스타일속성:값; }

        *해당 요소에 마우스가 올라가는 순간
        선택자:hover{ 스타일속성:값; }
