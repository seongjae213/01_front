font-size : 글자 크기 지정

        선택자{
            font-size: 크기(px, em(배수));
        }
  • 고정크기(px) 확인
  • 가변크기(em) 확인
  • 가변크기(%) 확인

font-weight : 글자 굵기 표현

        선택자 {
            font-weight: normal | bold | bolder | lighter | 100~900;
        }
  • 원래글자
  • 굵게 표현
  • 원래보다 굵게
  • 원래보다 가늘게

font-variant : 영문 텍스트 문구를 작은 크기의 대문자로 변경

        선택자{
            font-variant: small-caps;
        }

font-style : 글자를 기울여서 표현

        선택자 {
            font-style: normal | italic | oblique;
        }
em 태그
