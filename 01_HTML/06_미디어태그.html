이미지 태그 : img


* alt 속성 : 이미지 설명

이미지 경로가 잘못 되었을 경우 대체할 텍스트




* width, height 속성

이미지의 가로 길이(width), 세로 길이(height) 조정 가능
고정길이(px), 가변길이(%) 지정 => 기본값 : px

고정 길이(px) => 기본값(단위)

화면 크기가 변경되어도 이미지의 크기는 고정(변동 없음)





가변 길이(%)

화면 크기에 따라 이미지의 크기가 변경
부모 요소 크기에 따라서도 이미지의 크기가 변경

비디오 태그 : video
