    순서 있는 목록 태그

    기본적으로 1부터 시작 / type과 start로 표시 타입과 순서 정할 수 있음
    reversed로 역순 표시 가능
    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    숫자로 표시, 7부터 시작, 역순으로 목록 표시

    2. CSS
    3. JavaScript
    4. jQuery

    영문으로 표시, 7번째 알파벳 소문자로 시작하는 목록 표시

    3. JavaScript
