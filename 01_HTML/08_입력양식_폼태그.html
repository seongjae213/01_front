폼(form) 태그

form 태그 내에 submit 버튼 클릭 시 해당 form 태그 내에 작성된
사용자가 입력한 값들을 서버로 전달하면서 요청하는 역할을 수행

form action="" method=""

* action : 요청할 서버 주소
=> 해당 서버로 사용자가 form 내에 입력 태그들을 통해 작성한 값들을 전달
* method : 요청 전송 방식 지정 (get/post) 기본값 : get
=> get 방식 : 요청 시 사용자가 입력한 값들이 url에 노출되는 방식
=> post 방식 : 요청 시 사용자가 입력한 값들이 url에 노출되지 않는 방식

사용자가 입력한 값들은 key=value 형태로 전달

라디오 버튼과 체크박스 관련 input 태그의 타입

라디오 버튼

  • type="radio"
  • 제시한 값들 중 하나만 선택할 수 있도록 하고자 할 때
    • name 속성 값을 같도록 설정해야 함! => 요소끼리 하나의 그룹으로 묶여짐
  • form 전송 시 (submit) 현재 선택된 라디오 버튼의 value값이 전송됨
성별 :

체크 박스

  • type="checkbox
  • 제시한 여러 개의 값들 중 여러 개를 선택할 수 있도록 하고자 할 때
  • 선택된 체크박스의 value 속성 값이 전달됨 => 반드시 value 속성 값을 지정해야함!

그 외의 input 태그의 타입

type="color"

  • 색상을 선택할 수 있는 타입

type="file"

  • 파일을 첨부할 수 있도록 하는 타입

type="hidden"

  • 사용자에게 입력받지 않고 화면에 표시하지 않으며 요청 시 전달하고자 하는 값이 있는 경우


버튼 관련 타입들

  • 제출 : submit
  • 초기화 : reset
  • 일반 : button

select 태그, option 태그

  • 사용자가 직접 값을 입력하는 것이 아니라 제시한 여러 개의 값들 중 선택하도록 하고자 할 경우
  • 제출 시 선택된 option 태그의 값이 전달됨
    • 각 option 태그의 value 속성이 지정되지 않은 경우 content 영역의 값이 전달됨
    • value 속성이 지정되면 value 값이 전달됨
